# 🐍 Explorando Python con Ely Marquez

¡Bienvenido y bienvenida a este repositorio de aprendizaje! Este proyecto, creado por **Ely Marquez**, es una colección de ejercicios prácticos diseñados para explorar los fundamentos de la programación con Python de una manera divertida y progresiva.

Aquí no solo encontrarás código, sino una aventura de aprendizaje que va desde lo más básico hasta la creación de un pequeño sistema funcional.

---

## 🎯 Metas de Aprendizaje

- **Pensar como programador:** Desarrollar la lógica para resolver problemas.
- **Dominar los condicionales:** Hacer que los programas tomen decisiones.
- **Automatizar con ciclos:** Repetir tareas de forma eficiente con `for` y `while`.
- **Crear nuestras propias herramientas:** Escribir funciones reutilizables.
- **Organizar información:** Usar listas y diccionarios para manejar datos.
- **Construir un proyecto real:** Integrar todo lo aprendido en un programa de consola.

---

## 🗂 Mapa de la Aventura

El conocimiento está organizado en secciones, cada una construyendo sobre la anterior.

- **seccion_1_primeros_pasos**: ¡Hola, mundo! y operaciones básicas.
- **seccion_2_toma_decisiones**: Aplicando condicionales (`if`/`else`).
- **seccion_3_repeticion**: El poder de los bucles.
- **seccion_4_mis_propias_herramientas**: Creación de funciones.
- **seccion_5_organizando_datos**: Trabajando con estructuras de datos.
- **seccion_6_mi_proyecto**: **"Diario de Viajes Mágicos"** - Un sistema interactivo para gestionar un diario personal.

---

## ⚙️ Preparando el Equipaje

Para ejecutar estos programas, necesitas:

- **Python 3.10 o superior.** Si no lo tienes, descárgalo de [python.org](https://www.python.org/).
- Un editor de texto o IDE (Entorno de Desarrollo Integrado). Mis favoritos son:
  - **Visual Studio Code**
  - **PyCharm**
  - **Thonny** (ideal para empezar)

Para confirmar que Python está listo, abre tu terminal o línea de comandos y escribe:
